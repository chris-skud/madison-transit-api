parking

Overview

The parking API provides real-time parking data from both the City of Madison Parking Utility and the UW Transportation Service. In addition to lot availability, API consumers can include current or future special events applicable to a give lot which aides in detecting potential changes to lot's cost and payment policy.

URL

http://api.smsmybus.com/parking/v2/lots

Parameters

Name Value Required Description
expand specialevents No If "expand=specialevents" is included as a query string parameter, any current or future special event(s) that apply to a given lot will be embedded in the lot payload as members of the specialEvents collection.

GET http://api.smsmybus.com/parking/v2/lots

Event Information

For special event parking, ParkingStartTime and ParkingEndTime designate when pay-on-entry begin and end, respectively. Typically, pay on entry begins two hours before EventTime, and ends 5 minutes after EventTime.
